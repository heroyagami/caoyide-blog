医疗事故的定义#

根据《医疗事故处理条例》，医疗事故是指医疗机构及其医务人员在医疗活动中，违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，过失造成患者人身损害的事故。

医疗事故的分级#

根据对患者人身造成的损害程度，医疗事故分为四级：

级别 损害程度 示例
一级 造成患者死亡、重度残疾 手术导致死亡、植物人状态
二级 造成患者中度残疾、器官组织损伤导致严重功能障碍 一侧肾切除、双眼失明
三级 造成患者轻度残疾、器官组织损伤导致一般功能障碍 部分肢体功能障碍
四级 造成患者明显人身损害的其他后果 延长治疗时间、增加痛苦

医疗事故的构成要件#

1. 主体是医疗机构及其医务人员#

  • 医疗机构：取得《医疗机构执业许可证》的机构
  • 医务人员：依法取得执业资格的医护人员

2. 发生在医疗活动中#

包括诊断、治疗、护理、用药、手术等医疗行为。

3. 存在违法违规行为#

违反：

  • 医疗卫生管理法律（如《执业医师法》）
  • 行政法规（如《医疗事故处理条例》）
  • 部门规章
  • 诊疗护理规范、常规

4. 存在过失#

  • 疏忽大意的过失：应当预见而没有预见
  • 过于自信的过失：已经预见但轻信能够避免

排除故意：医务人员故意伤害患者的，构成故意伤害罪，不属于医疗事故。

5. 造成患者人身损害#

必须有实际的人身损害后果，包括：

  • 死亡
  • 残疾
  • 组织器官损伤
  • 功能障碍
  • 延长治疗时间
  • 增加痛苦

6. 因果关系#

医疗行为与损害后果之间存在因果关系。

不属于医疗事故的情形#

根据《医疗事故处理条例》第33条，有下列情形之一的，不属于医疗事故：

  1. 紧急避险：在紧急情况下为抢救垂危患者生命而采取紧急医学措施造成不良后果的
  2. 医疗意外：在医疗活动中由于患者病情异常或患者体质特殊而发生医疗意外的
  3. 现有技术局限：在现有医学科学技术条件下，发生无法预料或不能防范的不良后果的
  4. 无过错输血：无过错输血感染造成不良后果的
  5. 患方原因：因患方原因延误诊疗导致不良后果的
  6. 不可抗力：因不可抗力造成不良后果的

医疗事故鉴定#

鉴定机构#

医学会鉴定：

  • 设区的市级地方医学会
  • 省、自治区、直辖市地方医学会
  • 中华医学会（疑难、复杂、全国有重大影响的）

司法鉴定：

  • 具有医疗损害鉴定资质的司法鉴定机构
  • 适用于医疗损害责任纠纷诉讼

鉴定程序#

申请方式：

  1. 医患双方共同委托
  2. 卫生行政部门移交
  3. 法院委托（诉讼中）

提交材料：

  • 鉴定申请书
  • 病历资料（门诊病历、住院病历、检查报告等）
  • 医患双方陈述
  • 其他相关材料

鉴定过程：

  1. 审查材料
  2. 听取双方陈述
  3. 专家讨论
  4. 出具鉴定书

鉴定时限： 一般45日内出具鉴定书

鉴定内容#

  1. 是否构成医疗事故
  2. 医疗事故等级
  3. 医疗过失行为在损害后果中的责任程度
    • 完全责任
    • 主要责任
    • 次要责任
    • 轻微责任

赔偿项目与标准#

赔偿项目#

根据《医疗事故处理条例》第50条，医疗事故赔偿包括以下项目：

1. 医疗费 按照医疗事故对患者造成的人身损害进行治疗所发生的医疗费用计算，凭据支付。

2. 误工费

  • 患者有固定收入的：按因误工减少的固定收入计算
  • 收入高于医疗事故发生地上一年度职工年平均工资3倍以上的，按3倍计算
  • 无固定收入的：按医疗事故发生地上一年度职工年平均工资计算

3. 住院伙食补助费 按医疗事故发生地国家机关一般工作人员的出差伙食补助标准计算。

4. 陪护费 患者住院期间需要专人陪护的，按医疗事故发生地上一年度职工年平均工资计算。

5. 残疾生活补助费

  • 根据伤残等级，按医疗事故发生地居民年平均生活费计算
  • 自定残之月起最长赔偿30年
  • 60周岁以上的，不超过15年
  • 70周岁以上的，不超过5年

6. 残疾用具费 因残疾需要配置补偿功能器具的，凭医疗机构证明，按普及型器具的费用计算。

7. 丧葬费 按医疗事故发生地规定的丧葬费补助标准计算。

8. 被扶养人生活费

  • 以死者生前或残疾者丧失劳动能力前实际扶养且没有劳动能力的人为限
  • 按户籍所在地或居所地居民最低生活保障标准计算
  • 不满16周岁的，扶养到16周岁
  • 年满16周岁但无劳动能力的，扶养20年
  • 60周岁以上的，不超过15年
  • 70周岁以上的，不超过5年

9. 交通费 按患者实际必需的交通费用计算，凭据支付。

10. 住宿费 按医疗事故发生地国家机关一般工作人员的出差住宿补助标准计算，凭据支付。

11. 精神损害抚慰金

  • 按医疗事故发生地居民年平均生活费计算
  • 造成患者死亡的，赔偿年限最长不超过6年
  • 造成患者残疾的，赔偿年限最长不超过3年

赔偿计算示例#

案例： 张某因医疗事故导致八级伤残，住院治疗30天，误工3个月，月工资8000元。医疗事故发生地居民年平均生活费为30000元，职工年平均工资为60000元。

计算：

  1. 医疗费：50000元（凭票）
  2. 误工费：8000元 × 3 = 24000元
  3. 住院伙食补助费：50元/天 × 30天 = 1500元
  4. 陪护费：60000元 ÷ 365 × 30 = 4932元
  5. 残疾生活补助费：30000元 × 30年 × 30% = 270000元
  6. 交通费：1000元
  7. 精神损害抚慰金：30000元 × 2年 = 60000元

合计约：41万余元

维权途径#

途径一：协商解决#

医患双方自行协商，签订赔偿协议。

优点： 快速、便捷 缺点： 患者处于弱势地位，可能赔偿不足 建议： 协商前咨询专业律师

途径二：行政调解#

向卫生行政部门申请调解。

程序：

  1. 向医疗机构所在地县级卫生行政部门提出申请
  2. 卫生行政部门组织调解
  3. 达成调解协议的，制作调解书

途径三：医疗纠纷人民调解#

向医疗纠纷人民调解委员会申请调解。

特点：

  • 专业性：调解员具有医学、法律知识
  • 中立性：独立于卫生行政部门和医疗机构
  • 免费：不向当事人收取费用

途径四：民事诉讼#

向人民法院提起医疗损害责任纠纷诉讼。

诉讼时效： 3年，自知道或应当知道权利受到损害之日起计算

举证责任：

  • 患者证明：存在医疗关系、损害后果
  • 医疗机构证明：医疗行为与损害后果之间不存在因果关系、不存在医疗过错

诉讼流程：

起诉 → 立案 → 举证 → 质证（病历）→ 鉴定 → 开庭 → 判决

证据收集#

关键证据#

1. 病历资料

  • 门诊病历
  • 住院病历（入院记录、病程记录、手术记录、出院记录等）
  • 护理记录
  • 检查报告、化验单
  • 医嘱单

复印病历： 患者有权复印客观病历，封存主观病历。

2. 实物证据

  • 药品、医疗器械
  • 输液瓶、注射器
  • 病理标本

封存： 医患双方共同封存，由医疗机构保管。

3. 证人证言

  • 同病房患者
  • 家属
  • 医护人员

4. 鉴定意见

  • 医疗事故技术鉴定
  • 医疗损害鉴定
  • 尸检报告

律师建议#

发生医疗纠纷后的应对#

  1. 封存病历：第一时间要求封存全部病历
  2. 保存证据：保存药品、器械等实物证据
  3. 不要医闹：通过合法途径维权，避免触犯法律
  4. 及时咨询：尽早咨询专业医疗纠纷律师

选择维权途径#

  1. 损害轻微：可先尝试协商或调解
  2. 损害严重：建议直接诉讼，并申请法院委托鉴定
  3. 死因不明：建议进行尸检明确死因

注意事项#

  1. 诉讼时效：注意3年诉讼时效，及时主张权利
  2. 鉴定选择：医学会鉴定和司法鉴定各有利弊，根据情况选择
  3. 赔偿预期：医疗事故赔偿标准相对较低，做好心理准备
  4. 专业律师：医疗纠纷案件专业性强，建议委托专业律师

结语#

医疗事故纠纷涉及医学、法学多个领域，处理起来复杂而漫长。患者及家属在维权过程中要保持理性，通过合法途径主张权利。同时，也要注意收集和保存证据，必要时寻求专业医疗纠纷律师的帮助，以维护自身合法权益。
