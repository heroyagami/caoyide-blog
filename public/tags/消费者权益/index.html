合同里那些你看不懂的小字——最终解释权归本公司所有，有效吗？

小赵在健身房办了张年卡，签合同的时候销售催她快签，说都是标准文本不用细看。用了两个月，她发现合同中有一行小字：本会所保留根据经营需要关闭部分区域的权力，不另行通知不退还费用。 后来健身房把游泳池关了。小赵当初就是为了游泳池才选的这家。她找健身房理论，对方指着那行小字说你自己签的字。 小赵问我，签了字的合同是不是就得认。 不是。有些条款，你签了字也不一定认。 这就是格式条款的问题。格式条款是当事人为了重复使用而预先拟定、未与对方协商的条款。健身房的标准合同、银行的信用卡章程、保险公司的保单条款、APP的用户协议——都是格式条款。 法律对格式条款有三重限制。 第一，提供格式条款的一方有提示和说明的义务。对于免除或减轻自己责任的条款，必须用合理的方式提示对方注意，并按对方的要求予以说明。没有提示也没有说明的，对方可以主张该条款不成为合同的内容。健身房那行小字藏在合同中间，没有加粗没有单独提示，小赵可以主张这条根本没进入合同。 第二，不合理地免除或减轻自己责任、加重对方责任、限制对方主要权利的格式条款无效。最终解释权归本公司所有这条就属于典型——凭什么只有你有解释权？本质上是一方单方决定合同内容的权利，侵犯了对方的合同自由。 第三，对格式条款有两种以上解释的，应当作出不利于提供格式条款一方的解释。这条规则很有用——你跟商家对合同里的某句话理解不一样，法律按你的理解来。 小赵的案子，健身房那行小字因为没有合理提示和说明，不构成合同内容。健身房要么开放游泳池，要么退款。 日常签合同遇到格式条款，你最有力的武器就是这一句：你这条有歧义，按不利于你的解释来。很少有人知道自己手上这张牌，但它是民法典明确给你的。 本文作者：曹义德律师（华中科技大学法学硕士，前反贪检察官，执业律师） 法律咨询：18872778990（微信同号）

装修装了三分之一，老板跑了——剩下的钱和烂摊子怎么办

小孟买了套二手房，找了一家装修公司，全包十二万。签了合同，付了首期款八万。 水电改完了，地砖铺了一半，工人突然不来了。小孟打电话给老板，关机。打给工头，工头说老板欠他工钱，不干了。 小孟赶到公司地址，招牌还在，门锁着。问了隔壁商户，说半个月前就搬走了，每天都有人来找。 八万块打了水漂，家里是个半拉子工程，墙上的线槽还露着，卫生间瓷砖贴了一半。 装修公司跑路，分两步走。 第一步，止损。你现在最重要的是把装修做完，不要让损失继续扩大。找一家新的装修公司或者直接找工人，把剩下的活干了，保留新的合同和付款凭证。这笔新花的钱将来可以作为你的实际损失向跑路的公司索赔。 第二步，追钱。 装修属于承揽合同纠纷。公司单方面停止施工、卷款失联，构成根本违约。你可以要求解除合同，退回未施工部分的款项，并赔偿损失。 追钱的路径有三种： 一是民事诉讼。起诉装修公司，要求返还预付款并赔偿损失。关键是你要知道公司准确的名称、统一社会信用代码和法定代表人。这些在合同上应该有，没有的话去工商网站查。 二是刑事报案。如果装修公司收了钱根本没打算施工，收了预付款直接跑路，可能构成合同诈骗罪。合同诈骗的立案标准比较低，很多装修跑路的案子能走进刑事通道。 三是找消协和市场监督管理局投诉。装修属于消费行为，适用消费者权益保护法。市监局可以对跑路的公司进行行政处罚，吊销营业执照。 小孟走了第一条路。我帮他查了工商信息，这家公司还在业，法定代表人是外地人，名下没有财产。这意味着告赢了也可能执行不到钱。 但有一条路可以走：追加法定代表人为被执行人。 这家公司是一人有限责任公司。小孟付的八万块预付款，收款账户不是公司账户，是老板个人微信。这就是财产混同。我帮小孟把公司和法定代表人同时列为被告，主张人格混同，要求法定代表人承担连带责任。 老板最后被限制高消费，坐不了高铁飞机，主动联系了调解。返了五万，剩下三万分期。 小孟用这五万找了新工人把活做完了。 以后装修，记住三件事。 第一，钱别一次性给。按工程进度分期付款，水电做完给一笔，瓦工做完给一笔，油漆做完给一笔，验收合格再付尾款。预付款不要超过百分之三十。你手里的钱就是你对装修公司的牵制力。 第二，营业执照拍下来。合同上要有公司准确的全称和统一社会信用代码。将来你要告他，这两样信息必须准确无误。 第三，付款走对公账户。如果老板让你扫他个人微信，你留个心眼——将来他跑路了，这就是人格混同的证据。 本文当事人信息已脱敏处理。 本文作者：曹义德律师（华中科技大学法学硕士，前反贪检察官，执业律师） 法律咨询：18872778990（微信同号）

买到了假货，除了退货还能让他赔你三倍

阿花在网上买了一个名牌包，页面写着「专柜正品、假一赔十」，价格三千五。 到手之后，她越看越不对劲。走线歪的，皮料味道刺鼻，五金件颜色不对。找了个鉴定平台验了一下，假的。 她找商家理论，商家态度很好，说可能是仓库发错了，让退回来，包运费。 阿花问我，就这样退了？ 我说不退。你让他退一赔三。 阿花愣了一下，「什么叫退一赔三？」 消费者权益保护法有一条「退一赔三」的规定，很多人不知道：经营者提供商品或者服务有欺诈行为的，应当按照消费者的要求增加赔偿其受到的损失，增加赔偿的金额为消费者购买商品的价款或者接受服务的费用的三倍。增加赔偿的金额不足五百元的，为五百元。 翻译一下：商家骗你，你买了假货，可以要求退还货款，同时赔偿货款的三倍。如果三倍不满五百块，直接赔五百。 阿花这个包三千五，退一赔三就是：退货拿回三千五，再赔一万零五百。一共一万四。 「真的假的？」阿花不相信。 我说你按我说的做。 第一步，不要退货。退货了就没有证据了。保留这个包，保留所有包装和吊牌。 第二步，去那个鉴定平台出一份正式的鉴定报告，盖了章的那种，费用大概两三百。 第三步，在购物平台上申请售后，理由选「假冒品牌」，上传鉴定报告。平台一般会介入处理。 阿花照做了。平台客服介入后，商家主动联系她，说可以退款加补偿一千块，让她撤销投诉。 我说不要，坚持退一赔三。 最后平台判定商家售假成立，强制退款三千五，并从商家保证金里扣了一万零五赔给阿花。 阿花拿到钱的时候跟我说：「我以为买到假货只能自认倒霉。」 退一赔三有一个前提：商家构成欺诈。不是所有的质量问题都算欺诈。 什么叫欺诈？商家明知是假的，故意告诉你这是真的。比如明明是人造革说是真皮，明明是合成立方氧化锆说是天然钻，明明不是那个牌子却挂着那个牌子的标。 什么叫不构成欺诈？商家自己也不知道是假的，或者只是质量差但不是假冒品牌。比如一件衣服洗完缩水了，这叫质量问题，不叫欺诈。质量问题适用于七天无理由退货和三包规定，但不适用退一赔三。 还有一个关键：退一赔三针对的是生活消费。你买来自己用的、送给家人的，算。你买来再卖出去的，不算——那个走合同纠纷。 另外，食品和药品有更狠的规则：退一赔十。买到过期食品、假药劣药，赔偿金额是价款的十倍。 阿花后来跟我说，她把那个鉴定报告发到朋友圈了，好几个朋友告诉她，自己以前也买到过假货，都是退货了事，从来不知道还能赔三倍。 法律把权利放在那里，你不伸手，它就一直在那放着。 本文当事人信息已脱敏处理。 本文作者：曹义德律师（华中科技大学法学硕士，前反贪检察官，执业律师） 法律咨询：18872778990（微信同号）

网络购物消费维权指南

网络购物的法律依据 网络购物主要受《消费者权益保护法》《电子商务法》《民法典》等法律法规保护。 消费者的主要权利 1. 七天无理由退货权 适用范围： 消费者通过网络、电视、电话、邮购等方式购买的商品，有权自收到商品之日起7日内退货，且无需说明理由。 不适用七天无理由退货的商品： 消费者定作的商品 鲜活易腐的商品 在线下载或消费者拆封的音像制品、计算机软件等数字化商品 交付的报纸、期刊 其他根据商品性质并经消费者在购买时确认不宜退货的商品 退货条件： 商品应当完好 消费者应返还赠品 退货运费由消费者承担（另有约定除外） 2. 知情权 消费者享有知悉其购买、使用的商品或接受的服务的真实情况的权利。 商家应提供的信息： 商品或服务的真实、全面信息 不得作虚假或引人误解的宣传 不得虚构交易、编造用户评价 3. 安全保障权 消费者在购买、使用商品和接受服务时享有人身、财产安全不受损害的权利。 4. 公平交易权 消费者在购买商品或接受服务时，有权获得质量保障、价格合理、计量正确等公平交易条件。 5. 求偿权 消费者因购买、使用商品或接受服务受到人身、财产损害的，享有依法获得赔偿的权利。 常见纠纷及应对 一、收到假货 维权途径： 1. 平台投诉 向电商平台投诉商家售假 要求平台介入处理 要求退货退款并赔偿 2. 假一赔三 根据《消费者权益保护法》第五十五条，经营者提供商品有欺诈行为的，应当按照消费者的要求增加赔偿其受到的损失，增加赔偿的金额为消费者购买商品的价款的三倍；增加赔偿的金额不足500元的，为500元。 3. 假一赔十 根据《食品安全法》，生产不符合食品安全标准的食品或经营明知是不符合食品安全标准的食品，消费者除要求赔偿损失外，还可以向生产者或经营者要求支付价款十倍或损失三倍的赔偿金；增加赔偿的金额不足1000元的，为1000元。 4. 平台责任 电商平台明知或应知商家利用其平台侵害消费者合法权益，未采取必要措施的，依法与商家承担连带责任。 二、商品质量问题 维权步骤： 与商家协商：要求退货、换货或修理 平台介入：协商不成，申请平台客服介入 投诉举报：向市场监管部门投诉 起诉维权：损失较大的，向法院起诉 举证责任： 耐用商品（机动车、计算机、电视机、电冰箱、空调器、洗衣机等）6个月内发现瑕疵，由经营者承担有关瑕疵的举证责任 其他商品，消费者应证明商品存在质量问题 三、虚假宣传 常见形式： 夸大商品功效 虚构商品成分 使用虚假数据 盗用他人图片 刷单炒信 法律责任： 退货退款 三倍赔偿（欺诈情形） 行政处罚 四、拒不发货 商家责任： ...
