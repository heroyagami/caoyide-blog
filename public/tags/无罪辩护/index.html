无罪辩护

刑事案件的辩护策略

刑事辩护的基本原则 1. 以事实为依据，以法律为准绳 辩护律师应当依据案件事实和法律规定进行辩护，不歪曲事实，不曲解法律。 2. 维护当事人合法权益 辩护人的责任是根据事实和法律，提出犯罪嫌疑人、被告人无罪、罪轻或减轻、免除其刑事责任的材料和意见，维护其合法权益。 3. 独立辩护 辩护人依法独立进行辩护，不受当事人意志以外的其他因素干涉。 常见辩护策略 一、无罪辩护 适用情形： 案件事实不清、证据不足，或依法不构成犯罪。 常见类型： 1. 事实不清、证据不足 指控犯罪的证据不能形成完整证据链 证据之间存在矛盾不能排除 关键证据缺失 不能排除合理怀疑 2. 不在场辩护 提供不在犯罪现场的证据 证明案发时被告人在其他地方 3. 正当防卫 为了使国家、公共利益、本人或他人的人身、财产和其他权利免受正在进行的不法侵害 采取制止不法侵害的行为 对不法侵害人造成损害 4. 紧急避险 为了使国家、公共利益、本人或他人的人身、财产和其他权利免受正在发生的危险 不得已采取的紧急避险行为 所造成的损害小于所避免的损害 5. 未达刑事责任年龄 不满14周岁：不负刑事责任 已满14周岁不满16周岁：仅对特定重罪负刑事责任 不满18周岁：应当从轻或减轻处罚 6. 不具备刑事责任能力 精神病人在不能辨认或不能控制自己行为时造成危害结果 经法定程序鉴定确认 二、罪轻辩护 适用情形： 构成犯罪，但罪名认定有误，或存在从轻、减轻处罚情节。 1. 改变定性辩护 重罪改为轻罪 如：故意杀人罪 → 故意伤害罪 抢劫罪 → 抢夺罪 贪污罪 → 职务侵占罪 2. 从犯辩护 在共同犯罪中起次要或辅助作用 应当从轻、减轻处罚或免除处罚 3. 犯罪未遂 已经着手实行犯罪，由于犯罪分子意志以外的原因而未得逞 可以比照既遂犯从轻或减轻处罚 4. 犯罪中止 在犯罪过程中，自动放弃犯罪或自动有效地防止犯罪结果发生 没有造成损害的，应当免除处罚 造成损害的，应当减轻处罚 三、量刑辩护 适用情形： 对定罪无异议，但认为量刑过重，争取从轻、减轻处罚。 ...
