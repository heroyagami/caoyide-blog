有人可能会说：我捕了鱼，又放回去了，是不是就没事了？

2021年的吴成刚就是这么想的。他多次在长江重庆段——长江鲟的重要栖息地——使用渔网进行非法捕捞。2021年1月19日，他又一次下了网，结果捕到了3尾鲟鱼。

这次运气不好——其中一尾是长江鲟。

吴成刚知道这鱼不一般。他把鱼带回去之后没多久，渔政部门就找上门了。鱼被追回后放了生。

按理说，鱼都放了，损失应该不大吧？但检察院还是把他告了——而且不仅要求他赔偿生态损失，还要求他承担**惩罚性赔偿**。

这是怎么回事？

## 同一个案子，两个被告？

需要先厘清一下：本案和昨天讲的那个"吃了长江鲟"的案子（渝05民初37号）是**同一个系列**，但被告不同。

昨天的案子，被告是吴成刚、刘骄、蒋良聪、谢奇岐、章凡五人——谢奇岐和章凡是因为"吃"被追究责任。

今天的案子，被告是吴成刚一个人——他是实际捕捞者。

这两个案子的判决结果都发生在2022年7月27日，同一天出自同一家法院——重庆市第五中级人民法院。

## 案子的特殊之处：放生了还要赔钱？

吴成刚的辩护理由是：鱼已经放了，没有造成实质损害，为什么要赔偿？

法院的回答是：**放了不等于没损害。**

长江鲟是极危物种，被捕获时已经受到惊吓。更关键的是，法院请长江水产研究所的专家进行了评估：

长江鲟死亡个体具有较高的科研价值，其独特的个体属性是开展种群保护相关研究的重要素材。**即便放生后，长江鲟因受惊吓、饥饿等原因无法自然存活，也无法参与自然繁殖。**

所以，从生态保护的角度，这条被捕获的长江鲟，虽然形式上被"放生"了，但实际上已经永久失去了它的生态价值——它不会再参与繁殖，不会再为长江鲟种群的恢复做出贡献。

## 惩罚性赔偿是怎么算出来的

这是这个案子最精彩的部分。

根据《民法典》第一千二百三十二条，侵权人违反国家规定污染环境、破坏生态造成严重后果的，被侵害人可以请求相应的惩罚性赔偿。

但惩罚性赔偿具体怎么算？倍数是怎么确定的？

法院创立了一个很有意思的规则：**综合因素累计法**

法院把每一种可能影响惩罚性赔偿倍数的因素列出来，每个因素按0.1倍为标准进行加减：

| 因素 | 调整 |
|------|------|
| 侵权人以非法捕捞为职业 | +0.1倍 |
| 发生在禁渔区、禁渔期 | +0.1倍 |
| 使用禁用工具（渔网） | +0.1倍 |
| 捕获物已作放生处理 | -0.1倍 |
| 所获非法利益较大 | +0.1倍 |
| 已受刑事处罚 | -0.1倍 |
| 主动交纳生态修复金 | -0.1倍 |

把这些因素加起来，最终倍数不超过二倍。

本案中，法院以生态服务功能损失金额的**一倍**作为起始标准，然后综合上述因素，最终确定了惩罚性赔偿的倍数。

## 判决结果

1. 吴成刚赔偿生态服务功能损失93750元
2. 承担惩罚性赔偿责任
3. 按照鉴定报告的建议在非法捕捞水域放流成鱼、幼鱼进行生态修复
4. 如逾期未履行修复义务，则赔偿生态修复费用4966元

## 这个案例的重要意义

### 一、"放了生"不等于"没损害"

很多非法捕捞者有一个误区：只要我把鱼虾蟹放了，就不存在损害后果。但法院通过专家证言明确了一个科学事实：**对于濒危物种而言，被捕获这个行为本身就可能造成不可逆的损害。**

长江鲟是人工养殖和野生种群都极其稀少的物种。每一尾被捕获的长江鲟，无论最终是否存活，都已经失去了它的科研价值和繁殖价值。这个价值是无法用"鱼还活着"来衡量的。

### 二、惩罚性赔偿的计算方法创新

法院发明的"综合因素累计法"，既有一定的科学性，也有一定的灵活性。它不是简单的"一倍"或"两倍"，而是通过对各种因素的综合考量来确定一个合理的倍数。

这种计算方法未来可能成为类似案件的参考——通过对行为人的主观恶意、行为方式、损害后果等多个因素的综合打分，来确定一个公正的惩罚性赔偿数额。

### 三、生态修复不只是"还钱"

判决要求吴成刚在非法捕捞水域**实际放流成鱼和幼鱼**——这不是简单的赔钱了事，而是要求他**亲自参与生态修复**。

让侵权人亲手把自己的"错误"以实际行动弥补回来，比单纯的金钱赔偿更有教育和警示意义。
