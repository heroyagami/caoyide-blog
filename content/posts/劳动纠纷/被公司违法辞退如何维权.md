## 什么是违法辞退？

根据《劳动合同法》，用人单位解除劳动合同必须符合法定条件和程序，否则即构成违法解除。

### 合法解除的情形

**1. 协商解除（第36条）**
用人单位与劳动者协商一致，可以解除劳动合同。

**2. 劳动者过错解除（第39条）**
- 试用期不符合录用条件
- 严重违反规章制度
- 严重失职、营私舞弊，给单位造成重大损害
- 同时与其他单位建立劳动关系，严重影响本职工作
- 被依法追究刑事责任

**3. 无过错解除（第40条）**
- 患病或非因工负伤，医疗期满不能从事原工作也不能从事另行安排的工作
- 不能胜任工作，经培训或调岗仍不能胜任
- 客观情况发生重大变化，致使合同无法履行，协商变更不成

**4. 经济性裁员（第41条）**
需要裁减人员20人以上或不足20人但占职工总数10%以上，并履行法定程序。

### 违法解除的典型表现

1. **无正当理由辞退**：没有任何理由或理由不成立
2. **编造理由辞退**：以"不能胜任工作"等为由，但实际证据不足
3. **未履行法定程序**：如未提前30日通知、未支付代通知金
4. **辞退受保护员工**：如孕期、产期、哺乳期女职工
5. **未支付经济补偿**：解除合同时未依法支付经济补偿

## 违法辞退的法律后果

### 赔偿金标准

根据《劳动合同法》第87条，用人单位违法解除劳动合同的，应当按照经济补偿标准的**二倍**向劳动者支付赔偿金。

**计算公式：**
```
赔偿金 = 工作年限 × 月工资 × 2
```

### 经济补偿计算方法

**工作年限计算：**
- 每满一年支付一个月工资
- 六个月以上不满一年的，按一年计算
- 不满六个月的，支付半个月工资

**月工资标准：**
- 劳动者在劳动合同解除前12个月的平均工资
- 包括计时工资、计件工资、奖金、津贴、补贴等
- 高于当地社平工资3倍的，按3倍计算，且年限最高不超过12年

**举例：**
张某月工资8000元，工作3年7个月，被违法辞退。
赔偿金 = 4个月 × 8000元 × 2 = **64000元**

### 其他可能权益

1. **未休年假工资**：按日工资收入的300%支付
2. **加班费**：如有拖欠，应一并主张
3. **未签劳动合同二倍工资**：如存在未签合同情形
4. **补缴社保**：可要求补缴在职期间社保

## 维权流程

### 第一步：收集证据

**必备证据：**
- 劳动合同
- 工资流水/工资条
- 社保缴纳记录
- 工作证、工牌、考勤记录
- 辞退通知（书面、微信、邮件等）

**辅助证据：**
- 工作群聊天记录
- 工作成果文件
- 同事证言
- 录音录像

### 第二步：协商谈判

**谈判要点：**
1. 明确指出公司辞退行为的违法性
2. 提出合理的赔偿要求
3. 设定谈判期限，保留证据
4. 谈判过程可录音

**谈判不成怎么办？**
- 发送书面《解除劳动合同异议函》
- 向劳动监察部门投诉
- 准备申请劳动仲裁

### 第三步：申请劳动仲裁

**仲裁时效：** 自知道或应当知道权利被侵害之日起**1年内**

**申请材料：**
1. 仲裁申请书（一式三份）
2. 申请人身份证复印件
3. 被申请人工商登记信息
4. 证据材料及清单

**仲裁流程：**
```
提交申请 → 立案审查（5日）→ 开庭审理 → 作出裁决（45日，可延长15日）
```

**仲裁费用：** 免费

### 第四步：法院诉讼

**起诉条件：**
- 对仲裁裁决不服
- 仲裁委逾期未作出裁决

**诉讼流程：**
```
起诉 → 立案 → 开庭 → 一审判决（6个月）→ 上诉（15日）→ 二审判决
```

## 特殊保护情形

### 不得解除劳动合同的情形

根据《劳动合同法》第42条，劳动者有下列情形之一的，用人单位不得依照第40条、第41条解除劳动合同：

1. **职业病或工伤**：从事接触职业病危害作业未进行离岗前检查，或疑似职业病在诊断/医学观察期；在本单位患职业病或因工负伤并被确认丧失或部分丧失劳动能力
2. **患病或非因工负伤**：在规定的医疗期内
3. **三期女职工**：孕期、产期、哺乳期
4. **长期职工**：在本单位连续工作满15年，且距法定退休年龄不足5年

**注意：** 以上情形仅限制第40条（无过错解除）和第41条（经济性裁员），不限制第39条（劳动者过错解除）。

### 违法辞退三期女职工

**特殊保护：**
- 可以要求恢复劳动关系
- 或要求违法解除赔偿金
- 孕期、产期、哺乳期工资损失

**举例：**
李某孕期被违法辞退，月工资10000元。
- 违法解除赔偿金：工作2年，4个月工资 = 40000元
- 孕期工资损失：如孕期剩余5个月 = 50000元
- 产期工资损失：如产假128天 ≈ 42667元
- 哺乳期工资损失：如哺乳期剩余6个月 = 60000元
- **合计约19万余元**

## 常见误区

### 误区一：试用期可以随意辞退

**正解：** 试用期辞退也需要有合法理由，主要是"不符合录用条件"。用人单位需证明：
- 有明确的录用条件
- 已告知劳动者录用条件
- 劳动者确实不符合录用条件

### 误区二：签了离职申请就不能维权

**正解：** 如果离职申请是在胁迫、欺诈情况下签署的，或内容显失公平，可以主张撤销。

### 误区三：没劳动合同就没办法

**正解：** 没有书面劳动合同，可以凭工资流水、社保记录、考勤记录等证明劳动关系，还可主张未签合同的二倍工资差额。

### 误区四：公司倒闭就拿不到赔偿

**正解：** 公司破产、解散的，劳动者工资和经济补偿属于优先清偿债权。公司注销的，可追究股东责任。

## 律师建议

### 被辞退时的应对

1. **不要急于签字**：收到辞退通知不要马上签任何文件
2. **保留证据**：拍照、复印、录音，保存所有证据
3. **继续上班**：如公司未出具书面辞退通知，应继续正常上班
4. **咨询律师**：及时咨询专业劳动法律师

### 仲裁诉讼策略

1. **明确诉求**：是要求恢复劳动关系还是拿赔偿金
2. **计算准确**：准确计算各项赔偿金额
3. **证据充分**：准备充分的证据材料
4. **程序合规**：注意仲裁时效和诉讼程序

### 维权成本

**时间成本：**
- 仲裁：一般2-3个月
- 一审：一般3-6个月
- 二审：一般3个月

**经济成本：**
- 仲裁：免费
- 法院诉讼：10元
- 律师费：根据标的额协商，一般3000-20000元不等

## 结语

被违法辞退时，劳动者应当勇敢维护自身合法权益。通过合法途径维权，不仅可以获得应有的经济赔偿，也是对违法用人单位的有效制约。建议劳动者平时注意保存工作证据，了解劳动法规，必要时寻求专业律师帮助。
